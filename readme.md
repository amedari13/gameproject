Чтобы скопилировать проект

1. установить в папку external библиотеки:

 SFML-2.5.1
 Box2d

2. выполнить их построение, путём запуска скриптов

> build-box2d
> build-sfml-2.5.1

3. выполнить построение программы 

> build
